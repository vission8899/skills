# Skills Template

This directory contains templates and guides for creating new skills.

## Adding a New Skill

1. Copy `skill-template.md` to `<skill-name>/<skill-name>.md`
2. Fill in the template content
3. Add any supporting files (scripts, configs, etc.) in the skill directory

## Directory Structure

```
skills/
├── .skills/
│   ├── README.md          # This file
│   └── skill-template.md  # Template for new skills
├── skill-a/
│   ├── skill-a.md         # Skill content
│   └── ...
├── skill-b/
│   ├── skill-b.md
│   └── ...
└── README.md
```
