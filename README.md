# Skills Repository

AI coding assistant skills collection.

## Directory Structure

```
skills/
├── .skills/               # Templates and guides
├── <skill-name>/          # Each skill in its own directory
│   ├── <skill-name>.md    # Skill content
│   └── ...
└── README.md
```

## Adding a New Skill

1. Create a directory named after the skill
2. Create a markdown file with the same name
3. Follow the template in `.skills/skill-template.md`
